CSS-Relative-Units
  • rem = Font size of the root element px = Pixels (1 px = 0.75 pt)
  • % = Parent element dimensions cm = Centimeters
  • em = Font size of the element in = Inches
  • ex = x-height of the element's font pc = Picas (1 pc = 12 pt)
  • rlh = Line height of the root element
  • lh = Line height of the element pt = Points (1 pt = 1/72 in)
  • vb = 1% of viewport's size in the root element's block axis
  • vw = 1% of viewport's width
  • vh = 1% of viewport's height
  • vmax = 1% of viewport's larger dimension
  • vmin = 1% of viewport's smaller dimension
